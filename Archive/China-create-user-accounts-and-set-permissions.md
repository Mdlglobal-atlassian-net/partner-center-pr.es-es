<span data-ttu-id="58bb4-123">Acceso de centro de partners está basado en roles, lo que significa que puede restringir la vista de usuario para mostrar solo las características y funciones que un usuario necesita para completar tareas específicas.</span><span class="sxs-lookup"><span data-stu-id="58bb4-123">Partner Center access is role-based, meaning that you can restrict the user view to show only the features and functions that a user needs to complete specific tasks.</span></span> <span data-ttu-id="58bb4-124">Para cada usuario, tienes que seleccionar dos opciones:</span><span class="sxs-lookup"><span data-stu-id="58bb4-124">For each user, you must select two settings:</span></span>
