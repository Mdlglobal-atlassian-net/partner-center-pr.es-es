# <a name="buy-microsoft-azure-reservations-on-behalf-of-your-customers-in-partner-center"></a>Comprar reservas de Microsoft Azure en nombre de sus clientes en el centro de partners 

**Se aplica a**

-  Centro de partners
-  Microsoft Azure Portal
-  Partners de CSP

## <a name="before-you-begin"></a>Antes de comenzar

Revise la siguiente información importante antes de comprar las reservas de Azure en nombre de sus clientes.

-   Los clientes ya deben tener una suscripción de Azure activa para poder comprar reservas en su nombre.
  
-   No se incluyen los costos de suscripción de software, como el software de base de datos SQL o SUSE Linux en los precios de Azure de reserva

-   Precios comerciales de Microsoft le incluyen los impuestos, a menos que la ubicación es Brasil. Si la ubicación es Brasil, el precio comercial para incluye los impuestos adecuados 
 
-   Los agentes del departamento de soporte técnico y ventas necesitan acceso explícito a la suscripción de Azure para poder comprarla o administrarla en el Azure Portal y archivar solicitudes de soporte, incluidos los cambios y los reembolsos, en nombre del cliente.  

-   Si eres un proveedor indirecto y compras Azure Reservations a través del Azure Portal, el Partner en registro (revendedor indirecto) se hereda de la suscripción CSP de Azure que selecciones. 

-   El Partner en registro para Azure Reservations no se puede cambiar después de su adquisición. Puedes cancelar la reserva existente y comprar una nueva con el nuevo Partner en registro. 

-   Si un cliente quiere transferir una suscripción de Azure desde Direct o EA a CSP, no se transfieren las reservas.

## <a name="purchase-azure-reservations"></a>Adquirir reservas de Azure

Siga los pasos siguientes para comprar reservas de Microsoft Azure en nombre de sus clientes en el centro de partners.

1. Seleccione **clientes** en el menú del centro de partners.  

2. En tu página **Clientes**, busca el cliente que quiere comprar reservas de Azure y, a continuación, selecciona la flecha abajo para expandir la fila del cliente.  

3. Selecciona **Agregar productos** y, a continuación, **Azure**. 

    a. Elija el segmento de mercado del cliente desde el **segmento** lista.

    b. Elija **reservas** del producto **tipo** lista.

    c. Elija el tipo de reserva, el cliente espera el **tipo reservas** lista.

4. Las reservas de Azure deben estar asociadas con una suscripción de Azure activa. Elija la suscripción del cliente que desea agregar reservas de Azure a partir del **suscripción cliente** lista. 

   >[!IMPORTANT]
   >Si el cliente ya no tiene una suscripción activa de Azure, seleccione **servicios en línea** para agregar uno ahora. 

5. Use los filtros para buscar las reservas de Azure en máquinas virtuales que cumplen los requisitos de su cliente.  

6. Una vez encontrada la reservation(s) que desea comprar, escriba el número de instancias reservadas, el cliente deberá en **cantidad** y, a continuación, seleccione **agregar al carro**.  

7. Repite los pasos 5 y 6 hasta que hayas agregado todos los elementos necesarios al pedido. Selecciona **Revisar** para comprobar que el pedido es correcto.  

8. En la página **Revisar los pedidos**, puedes: 

    - Comprobar o cambiar la cantidad de instancias reservadas.

    - Seleccionar el ámbito de la reserva. El ámbito de la reserva puede abarcar una o varias suscripciones (ámbito compartido). Si establecer el ámbito de la reserva a una sola suscripción, el descuento de reserva se aplica a esta suscripción solo. Si seleccionas el modo compartido, el descuento de la reserva se aplica a las suscripciones del contexto de facturación del cliente. 

      >[!NOTE] 
      >Si decide para limitar el ámbito de la reserva para una sola suscripción de Azure, deberá aumentar la cuota de vCPU de la suscripción. Para aumentar la cuota de vCPU de la suscripción, deberá crear una solicitud de soporte técnico en Azure portal. Siga las instrucciones [en este tema](https://docs.microsoft.com/azure/azure-supportability/resource-manager-core-quotas-request) para crear la solicitud.    

    - Si eres un partner proveedor, selecciona el revendedor al que deseas asociar con el producto.

9. Selecciona **Comprar** para comprar el pedido. Los detalles de pedido, incluido el número de pedido, se muestran en el **confirmar** página. Selecciona **Listo** para ir a tu página **Historial de pedidos**. 

10. Para administrar la reserva del cliente en Azure portal, busque el cliente en su **clientes** página y, a continuación, seleccione la flecha abajo para expandir la fila del cliente. Selecciona **Portal de administración de Microsoft Azure** para abrir el registro del cliente en el portal de Azure.
