# <a name="partner-center-for-microsoft-cloud-germany"></a>Centro de partners para Microsoft Cloud Alemania

**Se aplica a**

-  Centro de partners para Microsoft Cloud Alemania

## <a name="microsoft-cloud-germany-meeting-german-data-protection-requirements"></a>Microsoft Cloud Alemania: Cumplimiento de los requisitos alemanes de protección de datos 

Microsoft Cloud Alemania está diseñada para cumplir los requisitos estrictos de protección de datos de Alemania e incluye un administrador de datos de confianza alemán, que se rige por la legislación alemana y que controla el acceso físico y lógico a los datos de los clientes. Todos los datos de los clientes y los sistemas que procesan o controlan el acceso a los datos de los clientes residen en centros de datos alemanes. La red que conecta estos centros de datos está aislada y dedicada en Alemania.

Microsoft Cloud Alemania proporciona una instancia independiente de [Microsoft Azure](https://go.microsoft.com/fwlink/?linkid=847992), Office 365 y Microsoft Dynamics, que se entregan desde centros de datos en Alemania. Como consecuencia, los clientes comerciales en la Unión Europea (UE) y la Asociación de Libre Comercio Europea (EFTA) pueden almacenar y administrar los datos de los clientes en cumplimiento de las leyes y normas alemanas vigentes, así como de los estándares internacionales más importantes. El personal de Microsoft no puede acceder a los datos de los clientes sin la aprobación y la supervisión del administrador de datos de confianza alemán o el cliente.

Para obtener más información, consulta la [hoja de datos de Microsoft Cloud Alemania](http://download.microsoft.com/download/6/1/3/613C9ECB-9167-4EF5-B131-3BAD8D8A126C/Microsoft_Cloud_Germany_Datasheet.pdf).

## <a name="frequently-asked-questions-about-the-csp-program-for-microsoft-cloud-germany"></a>Preguntas más frecuentes acerca del programa CSP de Microsoft Cloud Germany

Usa estas preguntas más frecuentes para obtener más información acerca del programa Proveedor de soluciones en la nube (CSP) de Microsoft Cloud Germany. También puedes ponerte en contacto con el administrador de cuentas para obtener más información. 

**¿Qué es Microsoft Cloud Germany?** 

Microsoft Cloud Germany combina una instancia de los servicios en la nube líderes del sector de Microsoft con una arquitectura de administración de sistemas única y una red dedicada en Alemania. En este modelo único, el administrador de datos, una compañía alemana independiente, controla el acceso a los datos de los clientes y no los comparte con terceros, excepto si lo indica el cliente o lo requiere la ley alemana. Esto también se conoce como soberanía de datos. 

**¿Quién puede participar en el programa CSP de Microsoft Cloud Germany?**

Los asociados de la Unión Europea (UE), de la Asociación de Libre Comercio Europea (EFTA) y del Reino Unido pueden participar en el programa CSP de Microsoft Cloud Germany y pueden vender a los clientes de la UE, de la EFTA y del Reino Unido. 

**CSP Commercial es un programa de asociados existente. ¿En qué se diferencia del programa CSP de Microsoft Cloud Germany?**

El programa CSP de Microsoft Cloud Germany es diferente del programa CSP Commercial en las siguientes áreas: 
-   Elegibilidad: el programa CSP de Microsoft Cloud Germany solo está disponible para asociados de la Unión Europea, la EFTA y el Reino Unido. 
-   Incorporación: los asociados deben incorporarse al programa CSP de Microsoft Cloud Germany independientemente del programa CSP Commercial. 
-   Licencias: los documentos de licencias del programa CSP de Microsoft Cloud Germany se actualizan para reflejar el rol del administrador de datos. A continuación se incluye información adicional acerca de los cambios en las licencias. 
-   Funcionalidad del Centro de partners: determinadas características del Centro de partners no se habilitarán para el programa CSP de Microsoft Cloud Germany para cumplir con los requisitos de soberanía de datos. Los asociados pueden usar el Portal de Microsoft Online (MOP), Azure Active Directory (Azure AD) o API Graph. 

**¿Cómo se diferencia la concesión de licencias del programa CSP de Microsoft Cloud Germany de la concesión de licencias de CSP Commercial?**

El contrato de revendedores de Microsoft Cloud (MCRA) y el contrato de distribuidores de Microsoft Cloud (MCDA) para el programa CSP de Microsoft Cloud Germany se diferencian del MCRA y del MCDA de Commercial de las dos siguientes maneras: 
-   Requieren que los clientes reciban el Contrato de Microsoft Cloud (MCA) para la nube alemana con un complemento para los términos de Online Services para la nube alemana 
-   Incluyen modificaciones en los requisitos de datos de clientes. El MCA para la nube alemana difiere del MCA de Commercial de las dos siguientes maneras: 
-   Incluye el complemento para los términos de Online Services, que incluye compromisos de protección de datos adicionales por parte de Microsoft para la residencia de datos, el rol del administrador de datos y el acceso a los datos del cliente 
-   Incluye el contrato del administrador de datos. El archivo de datos adjuntos 1 del complemento contiene un contrato entre el administrador de datos y el cliente, mediante el cual T-Systems se compromete directamente con el cliente respecto al control de sus datos y el acceso a ellos 

**¿Cuáles son las cualificaciones para vender el programa CSP de Microsoft Cloud Germany?**

Los criterios de cualificación para vender el programa CSP de Microsoft Cloud Germany son los mismos que para CSP Commercial, abierto para asociados directos y revendedores indirectos. No es necesario ser designado de nuevo para ser un revendedor indirecto del programa CSP de Microsoft Cloud Germany si estás aprobado como revendedor indirecto del programa CSP en Commercial. 

**Si ya soy asociado, ¿qué tengo que hacer para poder vender el programa CSP de Microsoft Cloud Germany?**

Los asociados deben incorporarse al programa CSP de Microsoft Cloud Germany. (La incorporación a Microsoft Cloud Germany es independiente de la incorporación a CSP Commercial). Una vez incorporado, podrás crear clientes y tramitar ofertas disponibles en Microsoft Cloud Germany. Si estás aprovechando las API, consulta más información a continuación sobre las API. 

**No soy partner de CSP todavía. ¿Dónde puedo encontrar más información acerca de cómo convertirme en partner de CSP?**

Visita el portal de MPN para obtener más información acerca de los modelos de inscripción disponibles para los asociados. 

**¿Los precios de cliente para el programa CSP de Microsoft Cloud Germany son los mismos que los de CSP Commercial?**

No. Los precios reflejarán costes adicionales para proporcionar el valor único de Microsoft Cloud Germany. 

**¿El modelo de descuento para asociados del programa CSP de Microsoft Cloud Germany es el mismo que el de CSP Commercial?**

Sí. El modelo de descuento para asociados del programa CSP de Microsoft Cloud Germany es el mismo que el de CSP Commercial. 

**¿Existe alguna diferencia entre los incentivos de los asociados del programa CSP de Microsoft Cloud Germany en comparación con CSP Commercial?**

No. Los incentivos para los partners del programa CSP de Microsoft Cloud Germany son los mismos que para CSP Commercial. Para obtener más información acerca de los incentivos de los asociados, visita el portal de MPN para obtener la información más reciente. 

**¿Existen cambios a la hora de facturar para el programa CSP de Microsoft Cloud Germany en comparación con CSP Commercial?**
No. Los requisitos y criterios de facturación del programa CSP de Microsoft Cloud Germany son los mismos que los de CSP Commercial. Para obtener más información acerca de la facturación, revisa la sección Recursos en el portal de MPN. 

**¿Son los requisitos de soporte técnico del programa CSP de Microsoft Cloud Germany los mismos que los de CSP Commercial?**

Sí. Para obtener más información acerca del soporte técnico, revisa la sección Recursos en el portal de MPN. 

**¿A qué segmentos de clientes puedo vender?**

Una vez que se te apruebe como asociado de CSP MCD, podrás vender a cualquier cliente comercial de la UE, la EFTA y el Reino Unido. 

**¿Qué productos y servicios están disponibles en el programa CSP de Microsoft Cloud Germany?**

En el momento del lanzamiento, estarán disponibles Microsoft Office 365 Germany y Microsoft Azure Germany. Microsoft Dynamics 365 Germany (CRMOL) está previsto para el segundo trimestre de 2017. También se agregarán otros servicios a medida que estén disponibles. 

**¿El programa CSP de Microsoft Cloud Germany ofrece todos los servicios de Azure?**

La infraestructura como servicio (IaaS), la plataforma fundamental y las ofertas de datos ya están disponibles. La guía básica se actualiza de manera continua y proporcionaremos documentación, como documentación de CSP Commercial, en el Centro de partners (se requiere autenticación).

**¿Existe una solución alternativa para las API que no están disponibles en el programa CSP de Microsoft Cloud Germany?**

Los asociados pueden usar el Portal de Microsoft Online (MOP), Azure Portal, Azure AD PowerShell o API Graph para la administración de cuentas de usuarios de asociados y la administración de licencias. Consulta el vínculo siguiente para obtener información técnica adicional: Developing with Partner Center for German Cloud (Desarrollar con el Centro de partners para la nube alemana) 

**En el caso de los datos hospedados en centros de datos de Microsoft de Dublín o Ámsterdam, ¿es posible cambiar a Microsoft Cloud Germany?**

Para facilitar y mantener la integridad del aislamiento de Microsoft Cloud Germany, no hay ningún mecanismo para mover el entorno de Microsoft Cloud existente de un cliente a Microsoft Cloud Germany. Si un asociado o cliente tiene un entorno en línea de Office 365, Azure o Dynamics CRM, será responsable de cualquier movimiento de datos a un nuevo entorno de Microsoft Cloud Germany.
